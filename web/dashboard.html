You can try the CRUD Endpoints! Didn't have time to built UI for it



To explore create, update, and delete operations, use Postman or any HTTP client:



  • GET /api/brands — list all brands
  • GET /api/brands/{id} — get a brand by ID
  • POST /api/brands — create a new brand
  • PUT /api/brands/{id} — update an existing brand
  • DELETE /api/brands/{id} — delete a brand
